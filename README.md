# 🛰️ Multi-Satellite Tracking & Collision Detection System

## 🚀 Overview
This project simulates and tracks multiple satellites using real-time TLE data.  
It visualizes satellite orbits in 2D and 3D and detects potential collisions.

## ✨ Features
- Real-time satellite tracking (TLE data)
- Multi-satellite visualization
- 3D orbit simulation
- Collision detection system
- Visual collision alerts

## 🛠️ Tech Stack
- Python
- Poliastro
- Plotly
- Dash
- SGP4
